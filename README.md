# ToDoList
ToDoList 📝
A clean, responsive To-Do List application built with Vanilla JavaScript. This project focuses on efficient task management and persistent data storage.

🚀 Features
Add Tasks: Quickly add new items to your list.

Mark as Complete: Toggle tasks to track your progress.

Delete Tasks: Remove completed or unwanted items.

Data Persistence: Uses Local Storage so your tasks remain even after refreshing the page.

Responsive Design: Optimized for both desktop and mobile viewing.

🛠️ Tech Stack
Frontend: HTML5, CSS3

Scripting: JavaScript (ES6+)

Storage: Browser LocalStorage API

📖 What I Learned
While building this mini-project, I strengthened my understanding of:

DOM Manipulation: Dynamically creating and removing elements based on user input.

Event Handling: Managing click events for multiple interactive elements.

State Management: Keeping the UI in sync with the data stored in the browser.
